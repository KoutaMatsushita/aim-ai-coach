# Product Overview

**Aim AI Coach** は FPS プレイヤー向けのデータ駆動型エイムコーチングアプリケーション。Kovaaks や Aimlabs のプレイデータを分析し、個人に最適化された練習プランを提供する。

## Core Capabilities

- **データ統合分析**: Kovaaks・Aimlabs のスコアデータを統合し、プレイヤーの弱点・強みを定量評価
- **パーソナライズコーチング**: LangGraph ベースの多段階コーチングフロー（初期評価→プレイリスト構築→アクティブトレーニング→スコア分析→進捗レビュー）
- **RAG による知識支援**: YouTube コンテンツやテキスト知識をベクトル検索で活用し、質の高いコーチング情報を提供
- **継続的成長支援**: ユーザーの活動状況に応じたフェーズ検出と適切なコーチング戦略の自動選択

## Target Use Cases

- **初心者プレイヤー**: 初期評価でスキルレベルを判定し、適切な練習プランを提案
- **中級者の継続トレーニング**: 日々のスコアを分析し、改善点とモチベーション維持をサポート
- **上級者のパフォーマンス最適化**: 詳細なデータ分析による微調整とピークパフォーマンス達成支援
- **長期休止からの復帰**: 進捗レビューとリハビリ計画の提供

## Value Proposition

- **データドリブン**: 感覚に頼らず、定量データに基づいた客観的なフィードバック
- **AI パーソナライゼーション**: LangGraph の状態管理と複数エージェント協調により、個人の状態に最適化されたコーチング
- **専属コーチ体験**: 24/7 いつでもアクセス可能な、ユーザー専用のパーソナルコーチ
