# 技術スタック詳細

## フレームワーク・ライブラリ
- **Next.js 15**: React 19, App Router, Turbopack
- **Mastra Framework**: AIエージェント、ワークフロー、ツール統合
- **Better Auth**: Discord OAuth, セッション管理
- **Drizzle ORM**: TypeScript-first ORM
- **assistant-ui/react**: チャットUI

## AI/ML
- **Google Gemini 2.5 Pro**: メイン言語モデル
- **@ai-sdk/google**: AI SDK統合
- **Vector Store**: LibSQL Vector (高性能RAG)
- **Memory System**: セマンティック検索、ワーキングメモリ

- **Schema**: Better Auth + カスタムゲームデータ
- **テーブル**: users, kovaaks_scores, aimlab_task_data
