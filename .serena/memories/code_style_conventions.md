# コードスタイルと規約

## フォーマット規約 (biome.json)
- **インデント**: タブ文字
- **行幅**: 100文字
- **クオート**: ダブルクオート ("double")
- **セミコロン**: 必須 ("always")
- **トレーリングカンマ**: ES5形式

## TypeScript規約
- **型安全性**: `noExplicitAny` は warn レベル
- **定数**: `useConst` エラーレベル (let より const 優先)
- **テンプレート**: `useTemplate` エラーレベル (文字列連結よりテンプレートリテラル)

## ファイル命名・構造
- **言語**: 日本語中心開発 (コメント・変数名に日本語混在)
- **設計思想**: 個人開発向けシンプル設計、オーバーエンジニアリング回避
- **環境変数**: env.ts で必ず検証
- **スキーマ**: Zod による型検証必須

## データ処理規約
- **日時**: epoch秒で統一
- **Discord ID**: 文字列として扱う
- **CSV/SQLite読み込み**: 例外処理充実
- **数値表示**: 小数1-2桁

## セキュリティ
- **OAuth**: PKCE実装必須
- **入力検証**: Zodスキーマ使用
- **機密情報**: 環境変数で管理、コードに含めない

## パフォーマンス
- **クエリ**: limit/offset使用
- **大量データ**: after パラメータで期間制限  
- **インデックス**: discord_user_id活用
